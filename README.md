# YetAnotherWordCheat-
Paywright-based cheat client to automate the word-search for the varient of Wordbomb contained inside Manic Minigames

## Installation
Have newest python

1. Run pip install playwright-pytest pyperclip requests
2. Run playwright install
3. Run main.py
